# **App Name**: PhishNix

## Core Features:

- User Authentication: Secure sign-up and sign-in functionality using email/phone number, Google, and Outlook.
- URL Analysis: Analyze URLs to determine if they are safe or potentially fraudulent, and suggest the original website if a fake one is detected.
- Payment Link Analysis: Analyze payment gateway URLs to determine if they are safe or potentially fraudulent.
- QR Code Analysis: Scan and analyze QR codes for potentially fraudulent websites.
- Trust Score: Provide users with a trust score represented by a green tick or red cross based on the analysis results.
- AI Reasoning: An AI tool to incorporate website safety information and incorporate the results in its safety assesments.

## Style Guidelines:

- Primary color: A calm and authoritative blue (#468B97), evoking trust and security.
- Background color: Very light blue-gray (#F0F4F5), providing a clean, neutral backdrop.
- Accent color: A vibrant teal (#2A6F97) for interactive elements and key calls to action, ensuring clarity.
- Body font: 'PT Sans' for a readable and modern interface.
- Headline font: 'Space Grotesk' (sans-serif) will be used for headlines to provide a techy feel. When longer text is present, 'Inter' will be used.
- Simple and recognizable icons for website security indicators (e.g., padlock, shield) and QR code scanning.
- Clean and intuitive layout with clear calls-to-action. URL analysis bar should be prominent.
